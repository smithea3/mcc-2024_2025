To construct the third-degree Taylor Polynomial for the function, it is perhaps best to consider a table of values that calculates \(\displaystyle f^{(k)}(c)\text{,}\) \(k!\) to easily determine the value of \(\displaystyle \frac{f^{(k)}(c)}{k!}\text{.}\) These will be the coefficients of the polynomial.
\(k\) \(f^{(k)}(c)\) \(k!\) \(\displaystyle \frac{f^{(k)}(c)}{k!}\)
\(0\) \(e^1\) \(1\) \(\frac{e}{1}\)
\(0\) \(e^1\) \(1\) \(\frac{e}{1}\)
\(0\) \(e^1\) \(2\) \(\frac{e}{2}\)
\(0\) \(e^1\) \(6\) \(\frac{e}{6}\)
Using this table, we can assemble the polynomial to be
\begin{equation*} P_4(x)=e+e(x-1)+\frac{e}{1}(x-1)^2+\frac{e}{6}(x-1)^3 \end{equation*}
