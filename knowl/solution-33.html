Solution 4.2.3.4.1.

Using the recursive definition of the factorial, we can simplify the summand as follows: \(\displaystyle \frac{k!}{(k+1)!}=\frac{k!}{(k+1)k!}=\frac{1}{k+1}\text{.}\) Using this information, we can rewrite the series as \(\displaystyle \sum_{k=0}^\infty\frac{1}{k+1}\text{.}\) This series is a candidate for the TheoremÂ 4.2.5: the Limit Comparison Test. Choose \(b_k=\frac{1}{k}\text{,}\) then
\begin{align*} \lim_{k\to\infty}\frac{a_k}{b_k} \amp =\lim_{k\to\infty}\frac{\frac{1}{k+1}}{\frac{1}{k}} \\ \amp = \lim_{k\to\infty}\frac{k}{k+1} \\ \amp = 1 \end{align*}
Therefore, since the limit is a positive real number, and the series \(\sum_k b_k\) diverges, then so does the given series.
