Solution 2.1.1.b.1.
Begin by writing the equation as
\begin{equation*} y'-y=4x\text{.} \end{equation*}
Now, we can identify the integrating factor \(\displaystyle\mu(x)=e^{\int p(x)\,dx}=e^{\int(-1)\,dx}=e^{-x}\text{,}\) where \(p(x)=-1\text{.}\)
Using the formula derived in class, we have
\begin{align*} y(x) \amp = \frac{1}{\mu(x)}\int \mu(x)f(x)\,dx\\ \amp = \frac{1}{e^{-x}}\int (e^{-x}(-4x))\,dx\\ \amp = -4e^x\int(xe^{-x})\,dx \\ \amp = -4e^x(-xe^{-x}-e^{-x}+C)\\ \amp = 4x+4+C_1e^x \end{align*}
Using the initial condition given, we find that \(C_1=-7e^{-1}\text{.}\) Therefore, out specific solution is \(\displaystyle\boxed{4x+4-7e^{x-1}}\) .
